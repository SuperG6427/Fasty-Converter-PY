# Fasty Converter

**Fasty Converter** es una herramienta de interfaz de línea de comandos (CLI) para conversion de imagenes.

## Funciones
Tiene detección automática de formatos y la manipulación de metadatos.
* **Conversión Multi-formato:** Soporta extensiones JPEG, PNG, WEBP, BMP, TIFF, RAW y HEIF.
* **Gestión de Calidad:** Compresión ajustables para formatos con pérdida como WEBP y JPEG.
* **Procesamiento por Lotes:** Selección granular o total de archivos dentro de directorios.
* **Validación de Dependencias:** El script verifica e instala automáticamente bibliotecas críticas mediante subprocesos de sistema.

## Uso
La interacción se realiza a través de un menú interactivo que guía al usuario en la configuración de rutas y parámetros.
1. **Ejecución:** Inicie el script con el comando `python fasty_converter.py`.
2. **Configuración de Rutas:** Defina el origen de los activos y el directorio de salida.
3. **Selección:** Marque los archivos deseados utilizando las herramientas de filtrado integradas.
4. **Ejecución:** Confirme los parámetros de calidad y metadatos para iniciar el proceso.
